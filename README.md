# DUNDETT

[![DUNDETT](https://img.shields.io/badge/DUNDETT-v1.3.4-brightgreen)](https://dundett.pages.dev)

A text adventure for the TI-84 Plus CE, inspired by Dunnet (the Emacs classic).

---

## WHAT IS DUNDETT?

You are at a dead end of a dirt road. The road goes east. 
In the distance you can see it will fork.

Explore caves. Befriend (or anger) a bear. Avoid the Grue. 
Find the office. Log into Windows XP. Wake up on Maple Street.

The game remembers. Sanity and death carry over. 
The Grue is watching.

---

## HOW TO PLAY

1. Download DUNDETT.8xp
2. Transfer to your TI-84 Plus CE
3. Press [prgm], select DUNDETT, press [enter]
4. Use menus to navigate. Enter codes when prompted.

---

## FEATURES

- 30+ locations across two major areas
- Persistent sanity and death system
- FUN value (1-100) changes secrets each game
- 5 collectible memory fragments
- Multiple endings
- Windows XP computer lab (yes, really)
- CAPTCHA verification
- Room 101 is mentioned. It is off limits.
